Variable Assignment

In Nulang, variable assignments are made using the - symbol. The syntax begins with a - followed by the variable's name and its assignment value. Example: -.1001./.1./ would assign the integer value 1 to the variable 1001 Additionally, all user-defined elements in Nulang, such as variables and functions, must be named using integers of 4 or more digits.

Whitespace

In Nulang, the "." character serves as a substitute for traditional whitespace and indentation. It functions as a separator for arguments as well as indicates indentations for loops and block structures.

Basic Operator Construct

In Nulang, all operators must be used in prefix notation. Example: ++.argument_1.argument_2 Logically equals: argument_1 + argument_2

Comparative Operators

++/ ... >, +// ... >=, --/ is <, -// is <=, /// is ==, and /-/ is !=.

Arithmetic Operators

++ ... + -- ... - // is / ** is *
