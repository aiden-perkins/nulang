About Nulang

Nulang is a programming language with the idea that you only need to use a computer's numpad, including its numbers and arithmetic symbols, to code. All functionality provided so far can be 100% utilized from the numpad, but the language can be used with a normal keyboard. So far, we can do:

  • If/Else If/Else Statements
  • Functions with parameters, pass by reference, & returning
  • Lists with get, set, and length
  • For & While Loops
  • Comparison Operators
  • Addition/Subtraction/Multiplication/Division
  • AND & OR operators
  • Lists, Integers, Strings
